# Solr
## Solr使用
### 基于Solr脚本启动
```bash
# 启动Solr服务
./solr start
# 停止Solr服务
./solr stop -all
```
此命令启动的是位于Solr解压根目录下 `/server/solr-webapp/webapp` 中的项目，启动所需的jar包在 `/server/solr-webapp/webapp/WEB-INF/lib` 中。
